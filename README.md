# TimeSeriesUnifiedToolbox

A unified framework for Time Series manipulation, forecasting, classification, and deployement, relying heavily on Ray capabilities and pipeline

## Error standardization suggestions

If you want to standardize exceptions across the library, see:

- `docs/ERROR_STANDARDIZATION.md`

## Contributing

Please install the necessary dependencies to contribute to this repository. You can do this by running the following command:

```bash
uv sync --dev

pre-commit install
```
